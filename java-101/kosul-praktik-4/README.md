# Koşul Praktik 4 - Aktivite Önerisi

Sıcaklığa göre etkinlik öneren Java programı.

## Kurallar

| Sıcaklık      | Öneri   |
|---------------|---------|
| < 5°C         | Kayak   |
| 5°C – 15°C    | Sinema  |
| 15°C – 25°C   | Piknik  |
| > 25°C        | Yüzme   |

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
