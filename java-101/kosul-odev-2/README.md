# Koşul Ödev 2 - Çin Zodyağı

Doğum yılına göre Çin Zodyağı burcunu hesaplayan Java programı.

## Formül

```
Doğum Yılı % 12 → Burç
```

## Burçlar

0=Maymun, 1=Horoz, 2=Köpek, 3=Domuz, 4=Fare, 5=Öküz, 6=Kaplan, 7=Tavşan, 8=Ejderha, 9=Yılan, 10=At, 11=Koyun

## Örnek Çıktı

```
Doğum Yılınızı Giriniz: 1990
Çin Zodyağı Burcunuz: At
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
