# Döngü Ödev 1 - En Büyük ve En Küçük Sayı

N tane sayı alıp en büyük ve en küçüğünü bulan Java programı.

## Örnek Çıktı

```
Kaç tane sayı gireceksiniz: 4
1. Sayıyı giriniz: 16
2. Sayıyı giriniz: -22
3. Sayıyı giriniz: -15
4. Sayıyı giriniz: 100
En büyük sayı: 100
En küçük sayı: -22
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
