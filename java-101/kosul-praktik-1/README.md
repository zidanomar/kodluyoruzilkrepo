# Koşul Praktik 1 - Hesap Makinesi (Switch-Case)

Switch-case kullanan basit hesap makinesi programı.

## Desteklenen İşlemler

`+`, `-`, `*`, `/`

## Örnek Çıktı

```
Birinci sayıyı giriniz: 10
İşlemi giriniz (+, -, *, /): *
İkinci sayıyı giriniz: 5
Sonuç: 50.0
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
