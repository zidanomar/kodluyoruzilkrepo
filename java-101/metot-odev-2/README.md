# Metot Ödev 2 - Eleman Frekansı

Dizideki her elemanın kaç kez tekrar ettiğini bulan Java programı.

## Örnek Çıktı

```
Dizi: [10, 20, 20, 10, 10, 20, 5, 20]
10 sayısı 3 kere tekrar edildi.
20 sayısı 4 kere tekrar edildi.
5 sayısı 1 kere tekrar edildi.
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
