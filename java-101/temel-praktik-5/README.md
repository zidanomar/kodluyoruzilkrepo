# Temel Praktik 5 - Daire Alan ve Çevre

Yarıçapı girilen dairenin alan ve çevresini hesaplayan Java programı. Ödev olarak daire dilimi alanı da hesaplanır.

## Formüller

```
Alan = π × r²
Çevre = 2 × π × r
Daire Dilimi Alanı = (π × r² × α) / 360
π = 3.14
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
