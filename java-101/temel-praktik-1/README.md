# Temel Praktik 1 - Not Ortalaması Hesaplayan Program

6 dersin sınav puanlarını kullanıcıdan alıp ortalamasını hesaplayan ve sınıfı geçip geçmediğini ekrana yazdıran Java programı.

## Dersler

Matematik, Fizik, Kimya, Türkçe, Tarih, Müzik

## Kurallar

- Ortalama > 60 → **Sınıfı Geçti**
- Ortalama ≤ 60 → **Sınıfta Kaldı**
- Ternary operator kullanılır

## Örnek Çıktı

```
Matematik puanını giriniz: 70
Fizik puanını giriniz: 80
Kimya puanını giriniz: 60
Türkçe puanını giriniz: 75
Tarih puanını giriniz: 65
Müzik puanını giriniz: 90
Ortalamanız: 73.33333333333333
Sınıfı Geçti
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
