# Koşul Praktik 3 - Not Ortalaması

5 dersin notunu alıp ortalama hesaplayan ve geçip geçmediğini gösteren Java programı.

## Dersler

Matematik, Fizik, Türkçe, Kimya, Müzik

## Kurallar

- Geçme notu: 55
- 0–100 aralığı dışındaki notlar ortalamaya katılmaz (ödev)

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
