# Döngü Praktik 6 - Armstrong Sayısı

Bir sayının Armstrong sayısı olup olmadığını bulan Java programı. Ödev olarak basamak toplamı hesaplanır.

## Armstrong Sayı Nedir?

N haneli bir sayının basamaklarının N'inci kuvvetleri toplamı sayının kendisine eşitse Armstrong sayısıdır.

**Örnek:** 407 = 4³ + 0³ + 7³ = 64 + 0 + 343 = 407

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
