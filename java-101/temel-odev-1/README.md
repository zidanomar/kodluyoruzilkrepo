# Temel Ödev 1 - Vücut Kitle İndeksi (VKİ)

Boy ve kilo değerlerine göre VKİ hesaplayan Java programı.

## Formül

```
VKİ = Kilo (kg) / (Boy (m) × Boy (m))
```

## Örnek Çıktı

```
Lütfen boyunuzu (metre cinsinde) giriniz: 1.72
Lütfen kilonuzu giriniz: 105
Vücut Kitle İndeksiniz: 35.49215792320173
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
