# Temel Praktik 4 - Taksimetre

Mesafeye göre taksi ücretini hesaplayan Java programı.

## Kurallar

- Açılış ücreti: 10 TL
- KM başına: 2.20 TL
- Minimum tutar: 20 TL

## Örnek Çıktı

```
Gidilen mesafeyi (km) giriniz: 5
Taksimetre Tutarı: 21.0 TL
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
