# Koşul Praktik 6 - Burç Hesaplayıcı

Ay ve gün bilgisine göre burcu bulan Java programı.

## Burçlar

Koç, Boğa, İkizler, Yengeç, Aslan, Başak, Terazi, Akrep, Yay, Oğlak, Kova, Balık

## Örnek Çıktı

```
Ayınızı giriniz (1-12): 3
Gününüzü giriniz: 25
Burcunuz: Koç
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
