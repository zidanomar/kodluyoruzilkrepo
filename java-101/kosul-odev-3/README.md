# Koşul Ödev 3 - Artık Yıl Hesaplayıcı

Girilen yılın artık yıl olup olmadığını bulan Java programı.

## Kural

- 4'e bölünebiliyorsa artık yıl
- 100'e bölünebiliyorsa artık yıl **değil**
- 400'e bölünebiliyorsa artık yıl

## Örnek Çıktı

```
Yıl Giriniz: 2020
2020 bir artık yıldır !

Yıl Giriniz: 1900
1900 bir artık yıl değildir !
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
