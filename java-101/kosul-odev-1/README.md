# Koşul Ödev 1 - Uçak Bileti Fiyatı

Mesafe, yaş ve yolculuk tipine göre uçak bileti fiyatı hesaplayan Java programı.

## Kurallar

- Km başına: 0.10 TL
- 12 yaş altı: %50 indirim
- 12–24 yaş: %10 indirim
- 65 yaş üstü: %30 indirim
- Gidiş-Dönüş: %20 ek indirim

## Örnek Çıktı

```
Mesafeyi km türünden giriniz: 1500
Yaşınızı giriniz: 20
Yolculuk tipini giriniz (1 => Tek Yön, 2 => Gidiş Dönüş): 2
Toplam Tutar = 216.0 TL
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
