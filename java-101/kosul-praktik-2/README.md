# Koşul Praktik 2 - Şifre Sıfırlama

Şifre yanlışsa kullanıcıya sıfırlama seçeneği sunan Java programı.

## Akış

1. Kullanıcı şifresini girer
2. Yanlışsa sıfırlama sorulur
3. Yeni şifre eski şifreyle aynıysa hata verilir, farklıysa başarılı mesajı gösterilir

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
