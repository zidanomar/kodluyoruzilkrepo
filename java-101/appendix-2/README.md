# Appendix 2 - Palindrom Kelime

Girilen kelimenin palindromik olup olmadığını bulan Java programı.

## Palindrom Nedir?

Tersten okunuşu aynı olan kelimeler: **abba**, **asa**, **kavak**, **kayak**

## Örnek Çıktı

```
Kelimeyi giriniz: kayak
kayak palindromik bir kelimedir.
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
