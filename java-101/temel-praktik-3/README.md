# Temel Praktik 3 - Hipotenüs ve Üçgen Alanı

Dik üçgenin hipotenüsünü ve Heron formülüyle üçgen alanını hesaplayan Java programı.

## Formüller

```
Hipotenüs = √(a² + b²)
u = (a + b + c) / 2
Alan = √(u × (u-a) × (u-b) × (u-c))
```

## Örnek Çıktı

```
Birinci dik kenarı giriniz: 3
İkinci dik kenarı giriniz: 4
Hipotenüs: 5.0
```

## Çalıştırma

```bash
javac src/Main.java
java -cp src Main
```
